# settings-configuration
## Environment Setup
Before running the tool you will need to setup the envrionment. All of the necessary packages are on the virtual environment included which must be activated every single time before running. To activate, run the command `source myenv/bin/activate` in the terminal. You should see (myenv) in the terminal before your computer's hostname.

## Running the tool
To run the tool type in `streamlit run settings_configs.py` in the terminal. This should bring a collection of links to open the tool in the web browser. Paste the local URL link or Ctrl click it to open the tool.
